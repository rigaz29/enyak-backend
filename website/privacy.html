Rembuk TV

Kebijakan Privasi

Terakhir diperbarui: 16 Juni 2026

Kebijakan Privasi ini menjelaskan data apa yang dikumpulkan aplikasi Rembuk TV ("Aplikasi"), untuk apa data tersebut digunakan, dan bagaimana kami memperlakukannya. Dengan menggunakan Aplikasi, kamu menyetujui kebijakan ini.

1. Data yang kami kumpulkan

Untuk mengelola akses donatur per perangkat dan menjaga keamanan layanan, kami mengumpulkan:

  • ID Perangkat (Android ID) — pengenal teknis perangkat, dipakai untuk mengaitkan akses donasi ke perangkat kamu.
  • Model perangkat (mis. merek & tipe HP) — untuk dukungan teknis dan diagnosa masalah.
  • Versi sistem operasi (versi Android).
  • Versi Aplikasi yang terpasang.
  • Alamat IP terakhir — untuk keamanan dan deteksi penyalahgunaan.
  • Status akses (trial / donatur / gratis) serta waktu aktivitas terakhir.

Kami tidak mengumpulkan nama, email, nomor telepon, lokasi GPS, daftar kontak, foto, atau data pribadi sensitif lainnya — kecuali informasi yang kamu kirim sendiri secara sukarela saat menghubungi admin lewat WhatsApp.

2. Bagaimana data digunakan

  • Mengaktifkan dan memvalidasi akses donatur pada perangkat kamu.
  • Memberikan dukungan teknis dan memperbaiki masalah.
  • Menjaga keamanan, mencegah penyalahgunaan, dan mendeteksi pelanggaran ketentuan.

3. Penyimpanan & pihak ketiga

Data disimpan di server kami (penyedia hosting). Kami tidak menjual data kamu. Data hanya dapat diakses oleh kami dan penyedia infrastruktur/hosting yang menjalankan layanan, sebatas yang diperlukan untuk mengoperasikan Aplikasi.

4. Masa simpan data

Data disimpan selama perangkat masih menggunakan layanan dan untuk jangka waktu wajar sesudahnya guna keperluan keamanan dan administrasi. Kamu dapat meminta penghapusan kapan saja (lihat bagian Hak kamu).

5. Hak kamu

Kamu dapat meminta untuk melihat atau menghapus data yang terkait dengan ID Perangkat kamu dengan menghubungi admin. Penghapusan data akan menonaktifkan akses donatur pada perangkat tersebut.

6. Anak-anak

Aplikasi tidak ditujukan untuk anak di bawah umur tanpa pengawasan orang tua/wali.

7. Perubahan kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Perubahan berlaku sejak dipublikasikan di halaman ini.

8. Kontak

Pertanyaan tentang privasi? Hubungi kami di rembuktv@protonmail.com atau WhatsApp admin.

Dokumen ini disusun sebagai informasi umum dan bukan nasihat hukum.
